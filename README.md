# CosPlix 🎭

A front-end web project for an anime cosplay community platform, built with HTML, Tailwind CSS, jQuery, and Lucide Icons.

---

## 📄 Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with animated showcase and hero section |
| About | `about.html` | Information about CosPlix and its mission |
| Events | `events.html` | Upcoming and past cosplay events (e.g. Comic-Con, Comiket, World Cosplay Summit) |
| Gallery | `gallery.html` | Photo gallery showcasing cosplay works |
| Community | `community.html` | Community hub for cosplay enthusiasts |
| Tutorials | `tutorials.html` | Cosplay guides and tutorials |
| News & Social | `news.html` | Latest news and social updates |
| Support | `contact.html` | Contact and support form |

---

## 🛠️ Technologies Used

- **HTML5** — Page structure and content
- **Tailwind CSS (v3.4.17)** — Utility-first CSS framework for styling
- **jQuery (v3.7.1)** — DOM manipulation and interactivity
- **Lucide Icons (v0.263.0)** — Icon library
- **Google Fonts** — Dela Gothic One & Outfit typefaces
- **Particles.js** — Animated particle background on the home page

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/cnicoke-sys/CosPlix.git
   ```

2. **Open in browser**
   - Simply open `index.html` in your browser, or
   - Use the **Live Server** extension in VS Code for a better development experience
